## Building
- Dependencies
`JDK17` and `Maven`.

That's it. No other bloat.

- Clone this repository
```sh
git clone https://github.com/clawbhaiya/ArruBot.git
```

- Compiling and Running
```sh
mvn compile
mvn clean install
mvn exec:java -Dexec.mainClass=arru.Arru
```
